Privacy Notes
A few practices that matter more than decorative security language.
  • Use aliases deliberately. Do not reuse public handles inside sensitive rooms.
  • Treat links carefully. Verify before opening unknown media or downloads.
  • Rotate credentials. Change room passwords if room membership changes.
  • Protect exported identities. Shared identity cards can be used to reconnect later.
